1. Hello

Aqui está um programa simples em Dart: 

void main() {
  print('Hello, World!');
}

Todo programa em Dart começa a execução pela função main().

Como existe uma chamada à função print() ela é executada.

A função print() imprime um objeto.

Para testar você pode acessar: https://dartpad.dev/?id=49cf09db06d2e98ff40ddcb37e2ea952

e apertar o botão > RUN

2. Variáveis

Podemos declarar uma variável usando var o tipo será determinado de acordo com o valor inicial. 

void main() {
  var mensagem = 'Olá, mundo!';
  print(mensagem);
  print(mensagem.runtimeType);
}

A propriedade runtimeType é uma representação do tipo do objeto.

Aqui vemos que runtimeType é uma propriedade que pertence ao objeto mensagem. Objetos possuem propriedades e métodos. As propriedades determinam o estado atual do objeto e os métodos determinam ações que o objeto pode realizar. 

void main() {
  var mensagem = 'Olá, mundo!';
  print(mensagem);
  print(mensagem.runtimeType);
}

3. Prática

A. No código anterior, use um método da variável mensagem para imprimir 'Olá, mundo!' com letras maiúsculas (UPPER CASE).

3.1. Solução

void main() {

4. Inteiros

Além de Strings, temos outros tipos, o int armazena um número inteiro. 

void main() {
  var ano = 2025;
  print('Ano: ');
  print(ano);     // teste runtimeType
}

Podemos usar valores de variáveis dentro de Strings usando $ antes do nome da variável: 

void main() {
  var ano = 2025;
  print('Ano: $ano');
}

Podemos indicar o tipo int: 

void main() {
  int ano = 2025;
  print('Ano: $ano');
}

6. Reais

Variáveis que armazenam números reais possuem o tipo double: 

void main() {

  double circ = 2 * raio * 3.14;

  print('Circunferência: $circ');
}

9. Coleções

O tipo List em Dart armazena vários elementos: 

void main() {
  var planetas = ['Mercúrio', 'Vênus', 'Terra', 'Marte'];

  print(planetas[0]);
  print(planetas.runtimeType);
}

Outra maneira de criar a lista é especificando seu tipo: 

void main() {
  List<String> planetas = ['Mercúrio', 'Vênus', 'Terra', 'Marte'];

  print(planetas[0]);
  print(planetas.runtimeType);
}

O número de objetos na lista fica na propriedade length: 

void main() {
  List<String> planetas = ['Mercúrio', 'Vênus', 'Terra', 'Marte'];

  print(planetas.length);
}

Não podemos ajustar um item além da quantidade (length): 

void main() {
  List<String> planetas = ['Mercúrio', 'Vênus', 'Terra', 'Marte'];

  print(planetas.length);

  planetas[5] = 'Júpiter';
}

A lista pode receber mais itens usando os métodos add e addAll: 

void main() {
  List<String> planetas = ['Mercúrio', 'Vênus', 'Terra', 'Marte'];

  planetas.add('Júpiter');

  print(planetas);
  print(planetas.length);
}

void main() {
  List<String> planetas = ['Mercúrio', 'Vênus', 'Terra', 'Marte'];

  planetas.add('Júpiter');    
  planetas.addAll(['Saturno', 'Urano', 'Netuno']);

  print(planetas);
  print(planetas.length);
}

11. Fluxo de controle

O comando if decide se vai ou não executar um bloco de códigos. 

if (ano >= 2001) {
  print('Século 21');
}

Um if/else executa um ou outro bloco de códigos: 

if (ano >= 2001) {
  print('Século 21');
} else {
  print('Antes');
}

É possível ter um if em um else: 

if (ano >= 2001) {
  print('Século 21');
} else if (ano >= 1901) {
  print('Século 20');  
} else {
  print('Antiguidade');
}

12. Prática

E. Altere o programa em Dart para mostrar se o mês atual é maior, menor ou igual ao mês informado, use if/else. 

void main() {
  // int informado = ?
  int atual = 10;

  print('$atual é maior que $informado');
}

12.1. Solução

void main() {
  int informado = 7;
  int atual = 10;

  if (atual > informado) {
    print('$atual é maior que $informado');
  } else if (atual < informado) {
    print('$atual é maior que $informado');
  } else {
    print('valores iguais: $atual');
  }
}

14. For

O comando for repete um bloco de comandos: 

void main() {
  int x = 5;

  for(var i = 0; i < x; i++) {
    print(i);
  }
}
